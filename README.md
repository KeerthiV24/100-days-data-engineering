# 🚀 100 Days Data Engineering Challenge

Welcome to my **100 Days Data Engineering Challenge**!

This repository documents my journey to become a **Data Engineer** by learning Data Engineering concepts from scratch and solving DSA problems daily using Python.

## 🎯 Goals

- Learn Data Engineering concepts step by step
- Master SQL and Python
- Solve 2 DSA problems every day
- Build consistency and improve problem-solving skills
- Share my learning journey on GitHub
